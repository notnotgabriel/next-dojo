# next-dojo

next.js + redux + redux-saga + react-testing-library + cypress

## Step by step guide

Run the commands
`npm init -y`
`npm install --save react react-dom next`
`mkdir pages`

add to package.json

```
"scripts": {
  "dev": "next",
  "build": "next build",
  "start": "next start"
}
```

Create a page at pages folder
Add a new page and links using `import Link from 'next/link';`
